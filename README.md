# Responsive Image Carousel 🖼️

An interactive and fully responsive photo slider component designed for modern web layouts, built using **Bootstrap 5** for design and native **JavaScript** for custom behavior controls.

## 🛠️ Key Features
- **Automated Transitions:** Seamless image rotation driven by background timing intervals (`setInterval`).
- **Interactive Controls:** Manual slide progression and user overrides mapped through custom event listeners.
- **Mobile-Responsive Design:** Fully fluid component styling built on top of the Bootstrap 5 grid and layout system.

## 🚀 Technologies Used
- **Logic & Control:** JavaScript (ES6+ / Vanilla JS)
- **UI Framework:** Bootstrap 5
- **Structure & Styling:** HTML5 / CSS3
- **Core Concepts:** Asynchronous timing functions, DOM property updates, and event-driven interfaces.
